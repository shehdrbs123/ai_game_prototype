# Concept One Pager

- Version: v0.1
- Last Updated: 2026-03-06
- Owner: Planning

## 목표 
- 출정 입력 구조
    - 제물 슬롯 3개 고정
    - 옥 슬롯 1개 별도
    - 옥은 0~1개만 넣을 수 있음
- 위험도 규칙
    - 위험도 명칭: 위험도 0, 1, 2, 3
    - 옥과 위험도는 1:1 매칭
    - 옥이 없으면 위험도 0
- 던전 결정 규칙
    - 각 제물 아이템은 미리 지정된 targetDungeon을 가짐
    - 제물 3개를 넣으면, 가장 많이 매핑된 던전으로 이동
    - 동률이면 seed 기반 랜덤(가중치 반영 가능)
- 랜덤 시드 규칙
    - 아이템 등급 계수 * 개수 총합을 기반으로 seed 생성
    - 제물 조합 문자열도 함께 반영해 재현성 확보
- 소모 정책
    - 출정 시 제물/옥 전량 소모
- 멀티플레이 고려
    - 장기적으로 가능성은 있으나, 현재 구현은 싱글 기준으로 진행
- 이벤트/운영 요구
    - 개발자 이벤트로 특정 던전만 뜨게 하는 강제 오버라이드 필요
    - 예: 이벤트 기간 forceDungeonId 지정
