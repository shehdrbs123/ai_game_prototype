# `CheatManager`

**파일 경로**: `src/managers/CheatManager.js`

## 역할
개발 및 테스트용 치트 콘솔 기능을 관리합니다.

## 주요 API
*   `toggleConsole()`: 콘솔 UI를 열거나 닫습니다. (기본 단축키: `)
*   **명령어**:
    *   `additem <itemId> [quantity]`: 인벤토리에 아이템을 추가합니다.
    *   `godmode`: 플레이어 무적 모드를 토글합니다.
    *   `help`: 사용 가능한 모든 명령어를 보여줍니다.
