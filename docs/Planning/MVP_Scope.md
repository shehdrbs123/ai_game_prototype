# MVP Scope

- Version: v0.1
- Last Updated: 2026-03-06
- Owner: Planning

## Scope Principle

- Must 중심으로 수직 슬라이스 완성
- Should/Could는 core loop 안정화 이후 단계적 반영

## Must

- Town / Dungeon / Result 상태 루프 완주
- 기본 전투(근접/원거리 포함)
- 루팅/인벤토리/장비 반영
- 제작 시스템 기본 레시피 동작
- 최소 1종 메타 업그레이드 루프

## Should

- 무기별 체감 차이 강화(리치/CD/피해 프로파일)
- 적 변주(행동 패턴 1~2종 추가)
- run 내 리스크 이벤트 포인트

## Could

- 특수 유물 효과(런 한정 modifier)
- 보상 선택 화면(2지선다 강화)
- 튜토리얼 경량 가이드

## Out of Scope (MVP)

- 실시간 멀티플레이(별도 `Multiplayer Strategy` 문서 기준으로 Post-MVP 진행)
- 대규모 서사/퀘스트 라인
- 복잡한 직업/스킬 트리

## Exit Criteria

- 신규 유저가 설명 없이 1 run을 완료 가능
- 핵심 루프 관련 치명적 버그(진행불가) 0건
- 초기 플레이테스트에서 재도전 의향 확인
