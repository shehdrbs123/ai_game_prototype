# Concept One Pager

- Version: v0.1
- Last Updated: 2026-03-06
- Owner: Planning

## 1) Game Pillar

- 장르: 2D Top-down Extraction Action
- 한 줄 정의: 던전에서 전투/루팅 후 탈출해 자원을 마을 성장에 투자하는 run-based action game
- 핵심 가치: 짧은 세션에서도 명확한 긴장(탈출 리스크)과 영구 성장(마을 메타)

## 2) Target Player

- 타깃: 1회 10~20분 세션을 선호하는 Action + 성장 루프 유저
- 플레이 판타지: 위험한 던전에서 생존해 가치 있는 전리품을 들고 복귀하는 생존자

## 3) Core Differentiator

- 차별점: 실시간 전투 숙련도와 마을 메타 성장의 결합
- 원칙: 기능 확장보다 core loop 완성도를 우선
- 확장 방향: Post-MVP에서 Co-op Multiplayer를 핵심 확장 축으로 적용

## 4) Core Loop (High-level)

1. Town 준비(장비/제작/업그레이드)
2. Dungeon 진입
3. 전투/탐색/루팅
4. 탈출 시도(리스크 관리)
5. 정산 및 성장 투자
6. 다음 run 준비

## 5) MVP Scope (Summary)

- 포함: Town-Run-Return 기본 루프, 무기 3종, 기본 제작/장비, 최소 메타 업그레이드
- 제외: 대규모 스토리, 클래스 다변화, 온라인 기능

## 6) Success Metrics (Draft)

- 10분 내 첫 run 완료율
- run 생존 복귀율
- 3회차 재도전율

## 7) Risks and Mitigation

- 리스크: 전투 피드백 부족 -> 대응: 타격감/SFX/VFX 우선 개선
- 리스크: 루프 단조로움 -> 대응: 보상 변주와 이벤트 포인트 추가
- 리스크: 메타 성장 체감 부족 -> 대응: 업그레이드 전후 효과 수치/연출 명확화
